## 📝 Project Overview
This project, conducted by **Group 11** from **Fulbright University Vietnam**, provides an in-depth data analysis and visualization of Bigfoot sightings across the United States. Through statistical analysis, we explore various aspects of the phenomenon, including geographic distribution, seasonal patterns, historical trends, and the impact of environmental factors (like weather and visibility) on sighting credibility.

**Instructor:** Prof. Tran Duy Hien  
**Teaching Assistant:** Tran Thanh Tung

## 📊 Dataset
The dataset used in this project contains detailed reports of Bigfoot sightings in the US, curated from the Bigfoot Field Researchers Organization (BFRO).
- **Author:** Tim Renner
- **Uploaded by:** The Devastator
- **Source:** Kaggle
- **File used in code:** `bfro_reports_geocoded.csv`

The data is categorized by sighting classifications (Class A, Class B, Class C) based on the type of evidence and credibility.

## 📂 Repository Structure
- `Midterm_Intro_To_Stats.ipynb`: The core Jupyter Notebook containing data exploration, data preprocessing, and statistical visualizations.
- `[Group 11] - Data presentation.pdf`: Presentation slides summarizing the project's key highlights, data presentation, and findings.
- `Group 11 - Data Infographic - Final.pdf`: A comprehensive and visually appealing infographic illustrating the project's conclusions.

## 💡 Key Insights
- **Geographic Distribution:** Sightings are heavily concentrated in specific regions, with the state of Washington reporting the most occurrences.
- **Seasonal Patterns:** Summer experiences the highest frequency of sightings compared to other seasons.
- **Weather Impact:** Environmental factors such as precipitation, cloud cover, wind speed, and humidity significantly reduce visibility, which impacts the overall credibility of the sightings.
- **Historical Trend:** Bigfoot sightings spiked notably during the 1970s and 1980s but have been on a steady decline in recent years.
